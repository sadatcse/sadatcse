## Chayatol Resort & Restaurant Management System
- Description: An all-in-one corporate administration and resource planning platform custom-tailored for Chayatol Resort & Restaurant. This unified software solution provides staff directory management, division organization, auditing logs, and robust role-based access controls (RBAC) powered by Next.js.
- Technologies Used:
  - FrontEnd: Next.js, React, DaisyUI, Tailwind CSS, Framer Motion, SweetAlert2, React Icons
  - Backend: Next.js (Server-Side Route Handlers), Mongoose
  - DataBase: MongoDB
  - Authentication: JWT, Bcrypt.js

## Teaxo POS
- Description: A state-of-the-art, feature-rich Point of Sale (POS) and comprehensive restaurant operation management client. It features real-time kitchen display synchronization using Socket.io, recipe/ingredient costing, vendor ledgers, granular role-based access control (RBAC), and AI-powered daily sales forecasting. Fully functional offline using IndexedDB (Dexie.js) and Progressive Web App (PWA) technologies.
- Technologies Used:
  - FrontEnd: React, React Router, Tailwind CSS, DaisyUI, Socket.io, Dexie.js (IndexedDB), Axios, Recharts, JSPDF, Moment.js
  - Backend: Node.js, Express.js
  - DataBase: MongoDB
  - Authentication: JWT, RBAC

## DataIT RX
- Description: A hybrid clinic management and prescription generation dashboard designed for doctors and clinical assistants. Built with an offline-first hybrid architecture, it uses Dexie.js (IndexedDB) with a serialized sync queue, client-side AES-256 cryptographic security to protect patient vitals and history transparently at rest, and dynamic action-level role-based permissions (RBAC).
- Technologies Used:
  - FrontEnd: React, React Router, Tailwind CSS, DaisyUI, Dexie.js, Crypto-JS (AES), Axios, Recharts, Moment.js
  - Backend: Node.js, Express.js
  - DataBase: MongoDB
  - Authentication: RBAC
